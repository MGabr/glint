class ResponseSerializer extends GlintSerializer

A fast serializer for responses

Internally this uses a very fast primitive serialization/deserialization routine using sun's Unsafe class for direct read/write access to JVM memory. This might not be portable across different JVMs. If serialization causes problems you can default to JavaSerialization by removing the serialization-bindings in the configuration.
