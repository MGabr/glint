AsyncBigWord2VecMatrix

Related Doc: package async

class AsyncBigWord2VecMatrix extends AsyncBigMatrixFloat with BigWord2VecMatrix

An asynchronous implementation of a BigWord2VecMatrix. You don't want to construct this object manually but instead use the methods provided in Client to run them on Spark, as so

Client.runWithWord2VecMatrixOnSpark(sc, bcVocabCns, vectorSize, n)
