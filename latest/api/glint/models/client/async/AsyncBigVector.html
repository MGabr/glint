abstract class AsyncBigVector[V, R, P] extends BigVector[V]

An asynchronous implementation of the BigVector. You don't want to construct this object manually but instead use the methods provided in Client, as so

The type of values to store

R

The type of responses we expect to get from the parameter servers

The type of push requests we should send to the parameter servers
