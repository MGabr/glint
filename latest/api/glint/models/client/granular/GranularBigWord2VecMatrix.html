GranularBigWord2VecMatrix

Related Doc: package granular

class GranularBigWord2VecMatrix extends GranularBigMatrix[Float] with BigWord2VecMatrix

A BigWord2VecMatrix whose messages are limited to a specific maximum message size. This helps resolve timeout exceptions and heartbeat failures in akka at the cost of additional message overhead.

The specific parameter server operations for efficient distributed Word2Vec training are not limited to a specific maximum message size. The operations for Word2Vec prediction are however.
